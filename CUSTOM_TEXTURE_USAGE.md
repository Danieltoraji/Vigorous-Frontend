# 自定义纹理功能使用说明

## 功能概述

在棋子编辑器的浮雕与纹理模块中，新增"自定义纹理"选项，允许用户使用灰度图生成浮雕效果。

## 使用步骤

### 1. 选择自定义纹理模式

1. 打开棋子编辑器
2. 选择底座或柱体组件
3. 在"浮雕与纹理"部分，将"形状"下拉框改为"自定义纹理"
4. 点击"🎨 配置自定义纹理"按钮

### 2. 选择纹理来源

在弹出的配置窗口中，有两个选项：

- **📂 从浮雕纹理管理器中选择**: 从已有的纹理资源中选择
- **✨ 生成浮雕**: 直接生成浮雕效果（推荐使用）

### 3. 选择纹理

1. 在纹理选择器中浏览可用的灰度图纹理
2. 点击纹理卡片进行选择
3. 点击"生成浮雕"按钮确认选择

### 4. 调整浮雕参数

选择纹理后，可以调整以下参数：

- **尺寸**: 控制浮雕的大小
- **深度**: 控制浮雕的高度（黑度映射为高度，白色高度为 0）
- **位置**: X、Y、Z 三个轴向的位置调整

## 技术实现原理

### 灰度图映射

系统使用 Three.js 的 displacement map 技术：

```javascript
<meshStandardMaterial
  displacementMap={textureLoader.load(textureFile)}
  displacementScale={depth}
/>
```

- **黑色区域** (RGB: 0,0,0) → 最大高度
- **白色区域** (RGB: 255,255,255) → 高度为 0
- **灰色区域** → 根据灰度值插值

### 纹理要求

为了获得最佳效果，建议使用：

1. **灰度图**: 黑白灰度分明的图片
2. **高分辨率**: 建议 512x512 或更高分辨率
3. **对比度高**: 黑白对比明显的图片效果更好

## 示例场景

### 创建文字浮雕

1. 使用图像处理软件创建黑底白字的图片
2. 上传到浮雕纹理资源管理器
3. 在棋子编辑器中选择该纹理
4. 调整深度和尺寸参数

### 创建图案浮雕

1. 创建黑色背景，白色图案的图片
2. 图案的灰度决定浮雕高度
3. 应用到棋子上，调整位置

## 注意事项

1. 确保纹理文件已经成功上传到浮雕纹理资源管理器
2. 自定义纹理需要一定的性能开销，建议在低面数模型上先测试
3. 如果浮雕效果不明显，可以尝试增加"深度"参数
4. 纹理的 UV 映射基于平面几何体，可能需要调整位置来对齐

## 故障排除

### 问题：浮雕不显示

**解决方案**：
- 检查是否选择了正确的纹理文件
- 确保 textureFile 字段已正确保存
- 尝试刷新页面重新加载数据

### 问题：浮雕效果太平

**解决方案**：
- 增加"深度"参数值
- 使用对比度更高的灰度图
- 检查纹理图片质量

### 问题：性能卡顿

**解决方案**：
- 降低纹理分辨率
- 减少 geometry 的分段数（64x64 是默认值）
- 使用更简单的模型
